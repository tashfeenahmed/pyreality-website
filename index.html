Data Scientist?

Rapidly visualise data in immersive environments

Prototyping

Rapidly create 3D immersive and stereoscopic data visualisations from your Jupyter Notebooks interface.

Interaction

Visualisations created with pyReality can be controlled from the Jupyter Notebooks interface while being viewed via an HMD.

Performance

Built with VRIA, D3.js and BabylonJs, pyReality utilises WebGL to render large datasets quickly and efficiently.
